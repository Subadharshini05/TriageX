triagex is a mobile-first AI-powered bug intelligence platform.

## Problem
Bug reports are often unclear, duplicated, and missing important details. This makes it hard for developers to understand and fix issues quickly.

## Solution
triagex helps engineering teams by using AI to:
- Understand messy bug reports
- Convert them into clear and structured issues
- Detect duplicate bugs early
- Make bug triage easier on mobile

## Target Users
- Developers
- QA testers
- On-call engineers

## Vision
triagex focuses on clean design, mobile usability, and AI-driven intelligence.

## Status
Project is in planning stage.
