## MVP Features (Hackathon Scope)

These features are included in the initial version of triagex and are mandatory for the product vision.

### Core Features
- Bug capture using free-text input
- Screenshot upload from mobile
- AI-powered bug structuring:
  - Title
  - Description
  - Steps to reproduce
  - Severity suggestion
  - Environment details
- Duplicate issue detection using vector search (RAG)
- AI preview before issue creation
- Simple triage board:
  - Open
  - In Progress
  - Resolved
- Issue detail view with AI insights

---

## Premium Differentiators
- AI insight cards (root cause hints, impact level)
- AI-assisted severity intelligence
- Crash awareness via Sentry integration

---

## Out of Scope (Intentional Exclusions)

The following features are intentionally excluded to maintain focus and quality.

- Sprint planning
- Story points
- Backlog grooming
- Team management & roles
- Analytics dashboards
- Release planning

---

## Scope Philosophy
triagex is designed to improve bug understanding and triage, not to replace full project management tools like Jira.
